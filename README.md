## ✨ Основные возможности

### 👤 Информация об игроке
- Детальная статистика с генерацией кастомной картинки-профиля
- Отображение никнейма, ID, уровня, административного статуса
- Даты регистрации и последнего входа
- Игровое время в часах
- Баланс виртуальной и донатной валюты
- Количество жетонов прокачки оружия и защиты уровня

### 🎮 Управление игроками (Action меню)
Интерактивная панель управления с возможностью:
- **Виртуальная валюта** — установка количества игровой валюты
- **Донатная валюта** — установка донатной валюты
- **Права администратора** — выдача/изменение уровня доступа
- **Социальный рейтинг** — изменение социального рейтинга (-1000 до 1000)
- **Никнейм игрока** — изменение никнейма
- **Пол игрока** — смена на мужской/женский
- **Жетоны прокачки оружия** — изменение количества
- **Жетоны сохранения уровня** — изменение количества
- **Удаление аккаунта** — полное удаление игрового аккаунта

### 🔗 Привязка аккаунта
- Привязка Discord аккаунта к игровому профилю MTA
- Защита от повторной привязки (привязать можно только один раз)
- Автоматическое начисление награды за привязку:
  - `15` жетонов прокачки оружия
  - `3` жетона сохранения уровня

### 🏠 Приватные голосовые комнаты
- Создание приватных голосовых каналов
- Управление комнатой через интерактивные кнопки:
  - **Скрыть/Открыть** комнату для всех
  - Изменение названия комнаты
  - Передача владения комнатой
  - Выгон участников
  - Выдача/забор доступа
  - Управление правами говорить
  - Изменение лимита пользователей
  - Открыть/закрыть комнату

### 🎁 Система промокодов
- Создание промокодов через модальное окно
- Настройка:
  - Название промокода (до 10 символов)
  - Количество использований
  - Сумма донатной валюты
  - Количество жетонов прокачки
  - Количество жетонов защиты
- Автоматическая рассылка уведомлений игрокам в личные сообщения

### 📊 Статус сервера
- Отображение информации о сервере:
  - Название игры и сервера
  - Режим игры и текущая карта
  - Версия сервера
  - Онлайн игроков
  - Ping до сервера
